# Marking rubric — Two Sum (10 marks)

- **Correctness (6 marks)** — returns the indices of a valid pair summing to the target
  on every test input.
- **Tricky inputs (3 marks)** — handles duplicate values, negatives, and the pair sitting
  at the two ends of the list.
- **Returns indices, distinct elements (1 mark)** — returns positions (not the values),
  and never uses the same index twice.
